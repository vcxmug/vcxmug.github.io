可信时间戳索引

每篇文章对应一个 RFC 3161 时间戳令牌（.tsr 文件），由 FreeTSA.org 签发。

如何验证

# 1. 获取原始 Markdown 文件（可从 blog-source 仓库获取）

# 2. 下载此目录下的证书
#    tsa.crt  — TSA 签名证书
#    cacert.pem — 根 CA 证书

# 3. 验证时间戳
openssl ts -verify -data post.md -in post.tsr \
  -untrusted tsa.crt -CAfile cacert.pem

# 输出 Verification: OK 即验证成功

证书文件

  • tsa.crt — TSA 签名证书
  • cacert.pem — 根 CA 证书
